# Folder Bookmarks

よく使うフォルダをサイドバーにブックマーク登録して、ファイルツリーをアコーディオンで展開できるミニExplorerです。

---

## 機能

- **ブックマーク登録**: 任意のフォルダをサイドバーに追加
- **アコーディオン展開**: クリックでファイルツリーを開閉（エクスプローラーと同じ感覚）
- **色分け**: ブックマークごとに8色から色を設定（デフォルト：黄色）
- **Finderで表示**: 右クリックメニューからFinderでフォルダを開ける

---

## 使い方

### フォルダを追加する

サイドバーの `+` ボタンをクリック → フォルダを選択

### フォルダを削除する

対象フォルダを右クリック → 「ブックマークから削除」

### 色を変更する

対象フォルダを右クリック → 「色を変更」→ 8色から選択

| 色 | カラーコード |
|---|---|
| 黄 | `#eab308` |
| 緑 | `#22c55e` |
| 青 | `#3b82f6` |
| 赤 | `#ef4444` |
| オレンジ | `#f97316` |
| 紫 | `#a855f7` |
| ピンク | `#ec4899` |
| グレー | `#6b7280` |
